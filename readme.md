# Cowork #

WordPress theme and functionality plugin for [Coworking Neuchâtel](https://coworking-neuchatel.ch).

## components

* [WordPress](http://wordpress.org/)
* Base theme: [Goran](https://wordpress.org/themes/goran/)
* Formidable (plugin)
* WooCommerce (plugin)
