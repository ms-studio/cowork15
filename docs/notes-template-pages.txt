IDEE: 

pour les pages WooCommerce, ajouter la classe no-sidebar-full:

add_filter( 'body_class', 'woo_cowork_custom_class' );

function woo_cowork_custom_class( $classes ) {
    if ( is_woocommerce() ) {
        $classes[] = 'no-sidebar-full';
    }
    return $classes;
}


is_woocommerce()
Returns true if on a page which uses WooCommerce templates (cart and checkout are standard pages with shortcodes and thus are not included).
